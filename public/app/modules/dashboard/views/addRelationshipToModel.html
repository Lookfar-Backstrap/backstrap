Entity Relationship

You are one click away from setting up a relationship. Optionally, you can add a "Rel Type" to this relationship. Enter the Rel Type value, or simply click the "Add Relationship" button.

What is a Rel Type?
Great question!! While the name property can be used as a value to search on, the Rel Type is used as a key to JOIN records together. If you are creating an address relationship for a peron's work, you would enter work. If you generate a query that joins on the rel_type Work, you would get ALL work addresses. This is not a requried field, but allows you to get the most out of Backstrap.
